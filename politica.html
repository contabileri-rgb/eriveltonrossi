Política de Privacidade

Última atualização: 12/03/2026

Esta Política de Privacidade descreve como são tratados os dados pessoais eventualmente fornecidos pelos usuários do site www.eriveltonrossi.com.br.

O site possui caráter institucional e informativo, apresentando informações profissionais e conteúdos relacionados à organização documental, temas administrativos e assuntos jurídicos.

1. Dados eventualmente tratados

O site não realiza coleta automática de dados pessoais. Dados podem ser fornecidos voluntariamente pelo usuário ao entrar em contato por:

  • E-mail
  • WhatsApp
  • Instagram

Esses dados podem incluir nome, telefone, e-mail ou outras informações fornecidas espontaneamente pelo próprio usuário.

2. Finalidade do tratamento

Os dados eventualmente recebidos são utilizados exclusivamente para:

  • Responder mensagens enviadas pelo usuário
  • Fornecer informações institucionais
  • Organizar comunicações e encaminhamentos informativos

3. Compartilhamento de dados

Os dados fornecidos pelos usuários não são comercializados nem compartilhados com terceiros, salvo quando necessário para cumprimento de obrigação legal.

4. Direitos do titular dos dados

Nos termos da legislação brasileira de proteção de dados, o titular pode solicitar informações sobre o tratamento de seus dados pessoais, bem como solicitar correção ou exclusão, quando aplicável.

Política de Privacidade

Última atualização: 12/03/2026

Esta Política de Privacidade descreve como são tratados os dados pessoais eventualmente fornecidos pelos usuários do site www.eriveltonrossi.com.br.

O site possui caráter institucional e informativo, com apresentação de perfil profissional, conteúdo relacionado à organização documental e informações de contato.

1. Dados eventualmente tratados

Este site não realiza coleta automática de dados pessoais por formulários próprios. Eventuais dados podem ser fornecidos voluntariamente pelo usuário ao entrar em contato por:

  • E-mail
  • WhatsApp
  • Instagram

Esses dados podem incluir nome, telefone, e-mail e outras informações enviadas espontaneamente pelo próprio usuário.

2. Finalidade do tratamento

Os dados eventualmente recebidos são utilizados para:

  • Responder mensagens enviadas pelo usuário
  • Prestar informações institucionais
  • Organizar comunicações e encaminhamentos informativos

3. Compartilhamento de dados

Os dados fornecidos não são comercializados e não são compartilhados com terceiros, salvo quando necessário para cumprimento de obrigação legal.

4. Direitos do titular dos dados

O titular dos dados pode solicitar informações sobre o tratamento de seus dados pessoais, bem como requerer correção, atualização ou exclusão, quando aplicável.
